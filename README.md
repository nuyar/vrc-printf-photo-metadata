# VRChat Printf Photo Metadata

[https://nuyar.github.io/vrc-printf-photo-metadata/](https://nuyar.github.io/vrc-printf-photo-metadata/)

## Variables

### Currently Supported
- `{worldName}`
- `{worldId}`
- `{worldUrl}`
- `{author}`
- `{authorId}`
- `{authorUrl}`
- `{YYYY}`, `{MM}`, `{DD}`, `{HH}`, `{mm}`, `{ss}`
- `{width}`, `{height}`

### Planned / Future Additions
- `{worldAuthor}`
- `{worldAuthorId}`
- `{worldAuthorUrl}`
